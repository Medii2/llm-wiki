# Datasets

This folder contains dataset pages.

Examples:
- Bundeshaushalt (gescraped)
- Förderdatenbank des Bundes (gescraped)
- Förderkatalog des Bundes
- SOEP
- UKHLS
- GHTorrent
- GH Archive

A dataset page should usually include:
- what the dataset covers
- geography and period
- unit of observation
- key variables
- strengths
- limitations
- likely use cases
- related projects and methods

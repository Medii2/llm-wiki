\# Project Notes



This folder contains project-local notes that are useful while the project is active.



Use this folder for:

\- temporary research notes

\- meeting notes

\- decision logs

\- work-in-progress thinking

\- draft summaries before they are cleaned up and written back to the global wiki



Rules:

\- Keep notes concise and structured.

\- Prefer one note per topic.

\- Do not treat this folder as the long-term knowledge base.

\- Durable knowledge should eventually be written back to the global research wiki.

\- If a note becomes stable and broadly useful, convert it into a proper wiki update.



Suggested note types:

\- `decision-log.md`

\- `data-issues.md`

\- `literature-notes.md`

\- `open-questions.md`

\- `next-steps.md`
